## 👋 안녕하세요! 

안녕하세요! 저는 **김기성**입니다. 🚀

백엔드와 프론트엔드를 넘나드는 풀스택 개발자로, **매일매일 새로운 기술을 배우고 기록하는 것**을 좋아합니다.

이 블로그는 제가 학습한 내용을 정리하고, 다른 개발자들과 지식을 공유하기 위해 만들었습니다.

### 📚 **매일매일 시리즈**
- **백엔드 기술일기** - Spring Boot, Java, 데이터베이스 등
- **프론트엔드 기술일기** - JavaScript, Vue.js, CSS 등

## 🛠️ 기술 스택

### 💻 **Backend**
- **Java & Spring Boot** - 메인 백엔드 기술
- **Spring Security** - 인증 및 권한 관리
- **JPA/Hibernate** - 데이터베이스 ORM
- **Redis** - 캐싱 및 세션 관리
- **MariaDB/MySQL** - 관계형 데이터베이스

- **JavaScript/TypeScript** - 프로그래밍 언어
- **CSS3/SCSS** - 스타일링 및 애니메이션
- **Bootstrap 5** - UI 컴포넌트 프레임워크

### 🚀 **DevOps & Tools**
- **Git & GitHub** - 버전 관리 및 협업
- **Docker** - 컨테이너화
- **Jenkins/GitHub Actions** - CI/CD 파이프라인
- **AWS/Cloud** - 클라우드 인프라

### 📋 **필수 요구사항**
- Ruby 3.2+ 
- Node.js 18+
- Git

### ⚡ **빠른 시작**

```bash
# 1. 저장소 클론
git clone https://github.com/saway126/saway126.github.io.git
cd saway126.github.io

# 2. 의존성 설치
bundle install
npm install

# 3. JavaScript 빌드
npm run build

# 4. 로컬 서버 실행
bundle exec jekyll serve

# 5. 브라우저에서 확인
# 로컬 개발 환경에서 확인
```
